Metodología BEM

La metodología Block, Element, Modifier (comúnmente conocida como BEM) es una convención de nomenclatura popular para clases en HTML y CSS.

Block Cualquier elemento autonomo que podemos reutilizar en cualquier parte de la página (menú, logo, galería, tarjeta).
A los bloques se les nombra con la funcionalidad del bloque.

< nav class="nav">
< header class="header">
< section class="container">
Element Un elemento son porciones más pequeñas internas a un bloque. Se usa para ir dividiendo el bloque en segmentos más pequeños.
Un elemento en BEM depende directamente de un Bloque, por lo que este debe estar dentro de un bloque.

< form class="form">
< input type="type" class="form__input">
< input type="submit" class="form__send">
< /form>
Modifier Un modificador sirve para modificar algunas propiedades de un bloque o elemento.
Un modificador en BEM, puede ser un bloque o un elemento, estos indican una modificación a dicho elemento o bloque.

< nav class="nav">
< a href="#" class="nav__link">
< a href="#" class="nav__link nav__link--disabled">
< /nav>

< input type="text" class="form__input form__input--color-gray">
